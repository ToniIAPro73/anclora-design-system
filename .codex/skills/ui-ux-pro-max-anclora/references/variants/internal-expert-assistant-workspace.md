# Variant: Internal Expert Assistant Workspace

Use when the product is an internal expert-assistance surface.

Reference:

- `Anclora Advisor AI`

## Protect

- obvious main interaction
- response-first hierarchy
- low cognitive friction
- specialized assistant tone

## Prioritize

- consultation clarity
- context framing
- next-step usefulness

## Watch for

- generic chat feel
- too much dashboard structure
- weak sense of domain expertise
