# Variant: Internal Executive Orchestration

Use when the surface should synthesize and coordinate at executive level.

Reference:

- `Anclora Command Center`

## Protect

- synthesis
- command hierarchy
- filtered, high-signal blocks
- few but meaningful actions

## Prioritize

- rapid executive reading
- cross-block coherence
- decision support

## Watch for

- generic analytics dashboard
- operational detail in first plane
- weak sense of control or orchestration
