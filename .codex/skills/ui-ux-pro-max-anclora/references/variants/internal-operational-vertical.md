# Variant: Internal Operational Vertical

Use when the surface is an internal operational workspace for a concrete vertical.

Reference:

- `Anclora Nexus`

## Protect

- speed of reading
- next-action visibility
- useful density
- tools-first composition
- focused navigation

## Current Nexus note

For `Nexus`, prefer the MVP reading:

- acquisition
- follow-up
- sellers
- buyers/leads
- properties
- tasks
- lightweight prospection

## Prioritize

- status readability
- task clarity
- list, filter, and form robustness
- navigation restraint

## Watch for

- too many modules in first plane
- public premium language leaking into daily ops
- expansion before operational discipline
