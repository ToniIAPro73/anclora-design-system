# Variant: Internal Ecosystem Gateway

Use when the surface acts as institutional entrypoint to the ecosystem.

Reference:

- `Anclora Group`

- institutional weight
- clear routing
- ecosystem order
- restrained corporate authority

- access clarity

- thin launcher feel
- not enough corporate gravity
