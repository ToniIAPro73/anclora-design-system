# Real Estate Profiles

Use this file for Real Estate surfaces.

## 1. Ultra Premium Core Platform

Reference:

- `Anclora Private Estates`

Protect:

- editorial precision
- luxury with control
- impeccable forms and overlays
- high-value CTA tone
- navigation that feels core, not ornamental

Common failure signs:

- luxury depends only on blur, gold, or image scale
- the app feels like stretched premium instead of ultra premium
- forms or navigation break under real use

## 2. Ultra Premium Market-Entry Landing

Reference:

- `anclora-private-estates-landing`

Protect:

- authority without fake inventory
- clear thesis from first viewport
- selective attraction for owners, investors, and partners
- visible connection to the ecosystem

Common failure signs:

- feels like a generic agency homepage
- simulates stock that does not exist
- mixes too many messages without a strong primary thesis

## 3. Premium Analytical

Reference:

- `Anclora Data Lab`

Protect:

- fast signal readability
- premium analytical framing
- controlled density
- strong shell and filter grammar

Common failure signs:

- generic BI dashboard feel
- decorative surfaces that hurt comprehension
- internal-operational density with no premium curation

## 4. Premium Relational

Reference:

- `Anclora Synergi`

Protect:

- trust
- selective admission
- continuity between public and private surfaces
- curated relationship tone

Common failure signs:

- generic partner form
- administrative portal feel
- weak sense of curation or status

## 5. Internal Operational Vertical

Reference:

- `Anclora Nexus`

Protect:

- speed
- visible priorities and next actions
- tools-first UX
- disciplined navigation
- low tolerance for modules that distract from daily work

Current reading for Nexus:

- treat the product as an MVP focused on acquisition and follow-up
- favor `sellers`, `buyers/leads`, `properties`, `tasks`, and lightweight prospection

Common failure signs:

- too many modules in first plane
- premium/public grammar leaking into operational work
- expansion before pipeline discipline
