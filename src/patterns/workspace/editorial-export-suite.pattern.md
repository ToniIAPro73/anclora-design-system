# Editorial Export Suite

## Objetivo

Patron canonico para pasos o paneles de exportacion editorial.

## Anatomia

1. `ac-export-suite`
2. `ac-export-suite__mark`
3. `ac-export-suite__content`
4. `ac-export-suite__title`
5. `ac-export-suite__summary`
6. `ac-export-suite__actions`

## Restricciones

- Usa `ac-button` para acciones y estados.
- No mezcla reglas de formato con la superficie visual.
- Los formatos concretos siguen siendo responsabilidad del consumidor.
