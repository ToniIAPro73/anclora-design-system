# Executive Patterns

Patrones para lectura de estado, resumen transversal y priorizacion de decisiones.

## Incluye

- bandas KPI
- cards de riesgo
- resumen de cobertura
- panels de drift o alertas

## No incluye

- dashboards cerrados a un único repositorio o equipo
- tablas o paneles construidos alrededor de un modelo de datos propietario
