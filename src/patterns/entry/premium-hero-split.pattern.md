# PremiumHeroSplit

## Objetivo

Combinar narrativa de marca y accion principal en una estructura de dos paneles reusable para Premium y Ultra Premium.

## Anatomia

- eyebrow
- title
- support copy
- action cluster
- support panel con meta o trust signals

## Slots

- `hero_identity`
- `hero_title`
- `hero_supporting_copy`
- `hero_actions`
- `hero_support_panel`

## Variantes

- `ceremonial`
- `operator-led`
- `portfolio editorial`

## Estados

- default
- reduced-motion
- compact

## Restricciones

- el patron define estructura y balance visual, no el storytelling final de una app
- si el panel derecho depende de widgets propietarios, eso pertenece a un example
