# Examples

Esta carpeta contiene referencias de consumo del sistema por tipo de app.

Un `example` puede ser especifico y cercano a producto. Su trabajo es demostrar adopcion, no definir canon.

## Tipos iniciales

- `premium/`
  Referencias para workspaces y productos de tono premium.
- `internal/`
  Referencias de herramientas operativas con mayor densidad y menor ceremonialidad.
- `portfolio/`
  Referencias editoriales y comerciales de portfolio.

## Regla rapida

- Si una composicion sirve en varias apps con estructura estable, promovela a `src/patterns/`.
- Si sirve para explicar una implementacion concreta, dejala aqui.
