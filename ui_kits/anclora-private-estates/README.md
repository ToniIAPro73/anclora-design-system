# Anclora Private Estates — UI Kit

Ultra-premium real estate landing. **Dark-only** per brand spec; gold (`#c9a95f`) on deep ink (`#0d1b1a`).

## Surfaces
- **Topbar** — pill nav with ES/EN/DE language switcher.
- **Hero** — italic serif display (`Newsreader`), gold eyebrow, gold metallic CTA, hero image panel with featured property caption + coin medallion overlay.
- **Metric strip** — four KPIs (portfolio, ticket, countries, operations).
- **Listings grid** — 3-up property cards with price, m², bedrooms, region eyebrow.

## Signatures
- Serif italic display with a highlighted word in gold.
- `radial + linear + grid-noise` triple-wash background.
- Rounded image containers with protection gradient at bottom.
- Gold button uses the metallic gradient (not flat gold).

Language switcher is cosmetic here; copy stays in Spanish for preview.
