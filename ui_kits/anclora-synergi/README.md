# Synergi — UI Kit

Partner workspace. Amethyst+copper palette on deep violet ink.

## Surfaces
- **Sidebar** — two-group nav (Workspace / Red) with active-state gradient dot.
- **Topbar** — breadcrumbs + user chip.
- **Hero card** — split: welcome copy + admission progress tracker.
- **KPI grid** — referrals, in-review, network conversion.
- **Referrals table** — recent referrals with status badges.

## Signatures
- Amethyst→copper gradient on primary button & active nav marker.
- Status badges: cyan (review), green (accepted), gold (submitted).
- Admission tracker uses numbered pill steps, check-marked when done.
